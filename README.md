# Weather

A web application that gives weather information from OpenWeather's API. Enter a city, press "Enter" on your keyboard, and you will get its weather information. It is designed to be as responsive as possible, so that it can be comfortably read on a mobile phone as well as on a PC.

Demo: https://weather-web-soumyabrata.vercel.app/

## Features
- Weather information fetched from OpenWeather's API.
- Current weather with temperature, air quality, humidity, pressure, visibiliy and wind speed.
- Five-day forecast.
- Themes: system default, dark and light.

## Tech Stack
- React
- Axios
- Tailwind CSS
- Vite
